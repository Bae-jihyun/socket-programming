# Python socket programming
파이썬을 이용한 소켓 프로그래밍
<br><br>

## Overview
Server가 제공하는 기능은 다음과 같습니다.
- 회원가입
- 로그인
- 영어 문자 Upper/Lower
<br>

## Guides
### 1. server 실행하기

```
python server.py
```

### 2. client 실행하기

```
python client.py
```

### 3. client 명령어 실행

- bye : 연결 종료

```
bye
```

- case : 소문자는 대문자로, 대문자는 소문자로 변환

```
case {english}
```

- signUp : 회원가입

```
signUp {ID}
```

```
{PW}
```

- signIn : 로그인

```
signIn {ID}
```

```
{PW}
```
